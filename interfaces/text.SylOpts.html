Optional sqnmlvy

sqnmlvy: boolean
  • Defined in text.ts:29

determines whether to regard the shewa under the letters שׁשׂסצנמלוי when preceded by a waw-consecutive with a missing dagesh chazaq as a shewa na'. If a metheg is present, the shewa is always a shewa na'.

defaultvalue

true

example
const default = new Text("וַיְצַחֵק֙");
default.syllables.map(syl => syl.text);
// ["וַ", "יְ", "צַ", "חֵק֙"]

const optional = new Text("וַיְצַחֵק֙", { sqnmlvy: false });
optional.syllables.map(syl => syl.text);
// ["וַיְ", "צַ", "חֵק֙"]

Optional wawShureq

wawShureq: boolean
  • Defined in text.ts:89

determines whether to regard a shewa after a vav-shureq as vocal. If a metheg is present, the shewa is always a shewa na'.

defaultvalue

true

example

default.syllables.map(syl => syl.text);

const optional = new Text("וּלְמַזֵּר", { wawShureq: false });
optional.syllables.map(syl => syl.text);
// ["וּלְ", "מַ", "זֵּר"]
remarks

results in example displayed in reverse order to mimic Hebrew writing; the rightmost value is the 0 item

Optional qametsQatan

qametsQatan: boolean
  • Defined in text.ts:106

converts regular qamets characters to qamets qatan characters where appropriate. The former is a "long-vowel" whereas the latter is a "short-vowel."

defaultvalue

true

example
const qQRegx = /\u{05C7}/u;
const default = new Text("חָפְנִי֙");
qQRegx.test(default.text);
// true

const optional = new Text("חָפְנִי֙", { qametsQatan: false });
qQRegx.test(optional.text);
// false
