hasVowelName

  • hasVowelName(name: keyof SyllableNameToCharMap): boolean
    • Defined in syllable.ts:163

    Returns true if syllables contains the vowel character of the name passed in

    According to Syllabification, a sheva is a vowel and serves as the nucleus of a syllable. Unlike Cluster, a Syllable is concerned with linguistics, so a sheva is a vowel character. It returns true for "SHEVA" only when the sheva is the vowel (i.e. a vocal sheva or sheva na').

    const text: Text = new Text("הַיְחָבְרְךָ");
text.syllables[0].hasVowelName("PATAH");
// true

// test for vocal sheva
text.syllables[1].hasVowelName("SHEVA");
// true

// test for silent sheva
text.syllables[2].hasVowelName("SHEVA");
// false

    Parameters

    • name: keyof SyllableNameToCharMap

    Returns boolean
