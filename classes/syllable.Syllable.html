vowel

  • get vowel(): null | (keyof SyllableCharToNameMap)

  • Returns the vowel character of the syllable

    According to {@page Syllabification}, a sheva is a vowel and serves as the nucleus of a syllable. Unlike Cluster, a Syllable is concerned with linguistics, so a sheva is a vowel character

    const text: Text = new Text("הַֽ֭יְחָבְרְךָ");
    text.syllables[0].vowel;
    // "\u{05B7}"
    text.syllables[1].vowel;
    // "\u{05B0}"

    Returns null | (keyof SyllableCharToNameMap)

    Description

    This returns a single vowel character, even for most mater lectionis (e.g. a holam vav would return the holam, not the vav). The only exception is a shureq, which returns the vav and the dagesh because there is no vowel character for a shureq.

hasVowelName

  • hasVowelName(name): boolean

  • Returns true if syllables contains the vowel character of the name passed in

    According to {@page Syllabification}, a sheva is a vowel and serves as the nucleus of a syllable. Unlike Cluster, a Syllable is concerned with linguistics, so a sheva is a vowel character. It returns true for "SHEVA" only when the sheva is the vowel (i.e. a vocal sheva or sheva na').

    const text: Text = new Text("הַיְחָבְרְךָ");
    text.syllables[0].hasVowelName("PATAH");
    // true

    // test for vocal sheva
    text.syllables[1].hasVowelName("SHEVA");
    // true

    // test for silent sheva
    text.syllables[2].hasVowelName("SHEVA");
    // false

    Parameters

    • name: keyof SyllableNameToCharMap

    Returns boolean

    Description

    This returns a boolean if the vowel character is present, even for most mater lectionis (e.g. in a holam vav construction, "HOLAM" would return true) The only exception is a shureq, because there is no vowel character for a shureq.

structure

  • structure(withGemination?): [string, string, string]

  • Parameters

    • withGemination: boolean = false

      If this argument is true, include gemination of the next syllable's onset in this syllable's coda.

      const text: Text = new Text("מַדּוּעַ");
      text.syllables.map(s => s.structure(true));
      // [["מ",
      //   "ַ",
      //   "דּ"],
      //  ["דּ",
      //   "וּ",
      //   ""],
      //  ["",
      //   "ַ",
      //   "ע"]]

    Returns [string, string, string]

    the structure of the Syllable, i.e. the syllable's onset, nucleus, and coda.

    • The onset is any initial consonant of the syllable - present in every syllable except those containing a except word-initial shureq or a furtive patah.
    • The nucleus is the vowel of the syllable - present in every syllable and containing its vowel (with any materes lecticonis) or a shureq.
    • The coda is all final consonants of the syllable - not including any matres lecticonis, and including the onset of the subsequent syllable if the subsequent syllable is geminated and the withGemination argument is true.
