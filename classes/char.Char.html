sequencePosition

  • get sequencePosition(): number

  • Returns number

    a number used for sequencing

    • consonants = 0
    • ligatures = 1
    • dagesh or rafe = 2
    • niqqud (i.e vowels) = 3
    • taamim (i.e. accents) = 4
    const text: Text = new Text("אֱלֹהִ֑ים");
    text.chars[0].sequencePosition; // the aleph
    // 0
    text.chars[1].sequencePosition; // the segol
    // 3
